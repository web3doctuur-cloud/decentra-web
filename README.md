# 🌐 DecentraWeb

> **Empowering the next generation of builders through decentralized education.**

DecentraWeb is a full-stack Web3 Learning Management System (LMS). It is currently in its **architectural phase** while I complete core certifications in Next.js 15 and Solidity development.

## 🏗 Planned Tech Stack
- **Framework:** Next.js 15 (App Router)
- **Language:** TypeScript
- **Styling:** Tailwind CSS
- **Smart Contracts:** Solidity (Hardhat)
- **Database/Auth:** Supabase
- **Web3 Connectivity:** Wagmi + RainbowKit
